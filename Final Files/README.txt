TRUE SIGNAL — LOGO FILES
========================
Brand: True Signal LLC · Consumer Strategy Advisory
Version: 1.0 · May 2026
Web: truesignalco.com

FILE TYPES
----------
.svg  Vector source. Text is outlined to paths (no font dependency). Use
      for web, social, and any modern design tool.
.png  Raster, 4× density, transparent background. Use for slides, screenshots,
      and any tool that doesn't accept SVG.
.pdf  Print-ready vector. Use for business cards, sell sheets, and any
      print application — scales to any size without loss.

LIGHT vs DARK
-------------
Light variants use Dark Navy (#1A2235) text/marks for placement on
white, cream (#F7F5F2), or other light grounds. The accent star/diamond
is in Burnt Orange (#C4622D).
Dark variants use Warm Off-White (#F7F5F2) text/marks for placement on
Dark Navy or other dark grounds; accent remains Burnt Orange.

COLORS
