LIGHT vs DARK
-------------
Light variants use Dark Navy (#1A2235) text/marks for placement on
white, cream (#F7F5F2), or other light grounds. The accent star is
in Burnt Orange (#C4622D).
Dark variants use Warm Off-White (#F7F5F2) text/marks for placement
on Dark Navy or other dark grounds; accent remains Burnt Orange.

NOTES
-----
All wordmark text is outlined to paths — the SVGs have no font
dependency and open correctly in any tool, on any machine.
All files have transparent backgrounds (except the social avatars,
which include their intended circular ground).
